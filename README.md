# 职场嘴替

一个飞书机器人，帮你用不同人格风格回复同事消息。

## 功能

- 职场人格测试（20道题，6种人格）
- 三种回复强度：温和版 / 标准版 / 火力版
- 逆袭体验：测试后可选择其他人格
- 外援功能：临时呼叫其他风格

## 六种人格

1. 🕊️ 和事佬·沙僧 - 附和即安全
2. 🔥 整顿侠·半泽直树 - 以牙还牙
3. 🌀 太极王·甄嬛 - 话不说满
4. 🧊 通透派·余华 - 幽默豁达
5. 🎭 捧哏王·岳云鹏 - 笑着糊弄
6. 💀 毒舌君·小S - 犀利吐槽

## 部署

1. 配置环境变量（复制 .env.example 为 .env）
2. 在飞书开放平台创建应用，配置事件订阅
3. 部署到 Render 或其他平台

## 飞书配置

- 事件订阅地址：`https://your-domain/`
- 权限：`im:message:receive`, `im:message:send`
